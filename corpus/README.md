# Corpus Sources

References to public Rego policy collections used to derive the shape taxonomy.
No policies are vendored — this directory tracks what was analyzed and how it was classified.

## Tracked Sources

- [open-policy-agent/gatekeeper-library](https://github.com/open-policy-agent/gatekeeper-library) — canonical Gatekeeper policies
- [open-policy-agent/contrib](https://github.com/open-policy-agent/contrib) — community-contributed OPA policies
